LaraPay payment integration for WooCommerce.

== Description ==

Allows customer to made payment on WooCommerce through LaraPay.

== Installation ==

1. Log in to your WordPress admin.
2. Search plugins "LaraPay for WooCommerce" and click "Install Now".
3. Activate the plugin.
4. Navigate to "LaraPay for WooCommerce" in the sidebar to access the plugin settings page.
5. Follow the instructions and update the plugin settings.
